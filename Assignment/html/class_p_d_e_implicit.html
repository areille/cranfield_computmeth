Detailed Description

A subclass of PDE Solver, specialized for implicit schemes.

The PDE implicit class provides:
-basic constructors for creating a Solver object with all initial and data (not meant to be used)
-a solve method which will solve the problem with all input values, for implicit methods.
-a virtual advance method which take to the next time and space step.
