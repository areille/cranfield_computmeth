Detailed Description

A subclass of PDE Implicit solver, specialized for Richardson scheme.

The Crank Nicholson class provides:
-basic constructors for creating a Solver object with all initial and data (not meant to be used) with Crank-Nicholson scheme.
-a advance method which take to the next time and space step.
