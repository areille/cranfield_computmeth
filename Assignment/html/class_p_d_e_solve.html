Detailed Description

A Solver class for one-dimensional PDE solving.
It has been designed only to solve one problem using differents . The problem is the evolution of the temperature a wall as the function of time.

The PDESolve class provides:
-basic constructors for creating a Solver object with all initial and data
-a solve virtual method which will differ if the scheme is or explicit
-a method which returns the results of the problems
